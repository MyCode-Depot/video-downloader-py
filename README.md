# YouTube Downloader GUI

Un téléchargeur de vidéos YouTube simple et moderne avec une interface graphique, développé en Python avec `CustomTkinter` et `yt-dlp`.

## Fonctionnalités
- Interface graphique moderne et épurée (CustomTkinter).
- Téléchargement automatique de la meilleure qualité vidéo et audio disponible (MP4).
- Intégration transparente de FFmpeg via `imageio-ffmpeg`.

## Prérequis

Avant de lancer l'application, assurez-vous d'avoir Python installé sur votre machine.

## Installation

1. Clonez ou téléchargez ce dépôt sur votre machine.
2. Ouvrez un terminal dans le dossier du projet et installez les dépendances requises à l'aide de la commande suivante :

```bash
pip install -r requirements.txt
